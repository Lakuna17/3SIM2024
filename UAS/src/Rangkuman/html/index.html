Belajar Web Development

1. Perbedaan HTML, CSS, dan JavaScript

HTML (HyperText Markup Language) adalah struktur dasar dari halaman web. CSS (Cascading Style Sheets) digunakan untuk memberikan gaya dan tata letak. JavaScript adalah bahasa pemrograman yang digunakan untuk memberikan interaktivitas pada halaman web.

Contoh Kerja Sama HTML, CSS, dan JavaScript

2. HTTP, GET dan POST

HTTP (Hypertext Transfer Protocol) adalah protokol yang digunakan untuk mentransfer data antara client (browser) dan server. Metode GET digunakan untuk meminta data dari server tanpa mengubah data, sedangkan POST digunakan untuk mengirim data ke server dan mengubah data tersebut.

3. Konsep DOM dalam JavaScript

DOM (Document Object Model) adalah representasi objek dari elemen HTML pada halaman web. Dengan DOM, kita dapat memanipulasi elemen-elemen HTML seperti menambah, mengubah, atau menghapus elemen.

4. Static Website vs Dynamic Website

Static website adalah website yang isinya tetap dan tidak berubah, biasanya menggunakan HTML dan CSS saja. Dynamic website menggunakan teknologi backend seperti PHP, Node.js, atau Python dan dapat menampilkan konten yang berubah berdasarkan input atau interaksi pengguna.

5. Cara Kerja Client-Server dalam Arsitektur Web

Client (seperti browser) mengirimkan permintaan HTTP ke server, yang kemudian memproses permintaan tersebut dan mengirimkan respons kembali ke client. Respons ini bisa berupa file HTML, gambar, atau data lainnya.
